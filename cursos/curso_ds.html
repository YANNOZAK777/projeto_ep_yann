Sobre o Curso

O curso de desenvolvimento de sistemas é voltado para pessoas que desejam aprender sobre programação e desenvolvimento de softwares. Os alunos aprenderão sobre linguagens de programação, estruturas de dados, algoritmos e outras áreas relacionadas.

Grade Curricular

Algumas das disciplinas que serão estudadas no curso de desenvolvimento de sistemas incluem:

  • Lógica de Programação
  • Algoritmos
  • Desenvolvimento de Software
  • Linguagens de Programação (Java, C++, Python)
  • Banco de Dados
  • Engenharia de Software
